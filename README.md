# 🎣 Phishing Awareness Training

A cybersecurity awareness project focused on educating users about phishing attacks, social engineering, and safe online practices.

---

## 📌 Project Overview

This project is a presentation-based training module designed to help individuals:

- Understand phishing attacks
- Identify fake emails and malicious websites
- Recognize social engineering tactics
- Apply best practices to stay secure online

---

## 🎯 Key Topics Covered

- What is Phishing?
- Types of Phishing (Email, Spear, Smishing, Vishing)
- Social Engineering Techniques
- Identifying Phishing Emails
- Detecting Fake Websites
- Real-world Cyber Attacks
- Security Best Practices
- Interactive Quiz

---

## 🧠 Learning Objectives

- Identify suspicious emails and links
- Understand attacker techniques
- Improve cybersecurity awareness
- Make safer online decisions

---

## 📊 Features

- Structured training slides
- Real-world phishing examples
- Scenario-based learning
- Interactive quiz section
